# 坦克大战 - Tank Battle

一个基于 HTML5 Canvas 和 JavaScript 的经典坦克大战游戏。

## 在线试玩

访问 [GitHub Pages](https://emma943983.github.io/tank-battle/) 即可在线游玩。

## 游戏特性

- 经典坦克大战玩法，保卫基地，消灭所有敌人
- 3个精心设计的关卡，难度递增
- 智能敌人AI，会追踪玩家并自动射击
- 多种地形：砖墙（可破坏）、钢墙（不可破坏）、水、草丛
- 精美的爆炸粒子特效
- 坦克履带动画效果
- 计分系统和生命值系统

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格键 | 射击 |
| P | 暂停/继续 |

## 本地运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 或者使用简单的HTTP服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- 无外部依赖
